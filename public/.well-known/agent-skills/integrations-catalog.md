# Last Mile Integrations Catalog

Use this skill when you need machine-readable discovery of Last Mile's public integration coverage.

## Endpoints

- Human-readable catalog: https://lastmileinc.ai/integrations
- Machine-readable catalog JSON: https://lastmileinc.ai/integrations.json
- Detail page pattern: https://lastmileinc.ai/integrations/{vendorSlug}/{productSlug}

## What the catalog contains

- Industries
- Vendors
- Products
- Detail page availability
- Integration API URLs where available

## Recommended use

- Discover supported vendors and products before proposing an implementation.
- Pull the JSON catalog when an agent needs structured inputs.
- Use detail pages for product-level coverage summaries.
